用户与租

useradd xxx 新建用户
passwd 修改用户密码
su 切换用户
userdel 删除用户（如果删除不了，find / -name 111 rm -rf 111 cd /var/spool/mail rm -rf 111） userdel -r 全部删除

home下可以查看用户 cat/etc/passwd/ 存储用户信息
第3位
uid唯一标识0为最高权限
•在centOS7中UID为1-999时为虚拟账户，不需要登录，傀儡账户。普通用户1000起，超级用户是0
•在centOS6中UID为1-499时为虚拟账户，不需要登录，傀儡账户。普通用户500起，超级用户是0
第4位是组gid

linux网卡配置

pwd：当前默认的用户目录
sudo：以管理员身份运行
cd：进入一个文件夹
su：切换到管理员账户
rm -rf ：删除所有文件
mkdir：创建文件夹
cd ../：返回上一目录
vim：创建新文件 vim/vi+文件后缀 生成一个新文件并编辑 i：插入编辑 w：保存 q：退出 ！：强制执行
clear 或者ctrl+l：清屏
cat：查看当前文件内容
ifconfig：查看当前网卡信息=ipconfig
centos7网卡配置文件：/etc/sysconfig/network-scirpt/ifcfg-enoxxxxxx 修改BOOTPROTO=DHCP为BOOTPROTO= none/
没有网络/ static/静态ip/
指定一个ip：IPADDR=192.168.1.xxx 网卡是否开机启动：ONBOOT
是否接受DHCP服务器指定的DNS地址：PEERDNS
设置网关必须与ip一个网段： GATEWAY=192.168.1.1
NETMASK=255.255.255.0
DNS1=”192.168.1.1”
systemctl restart network 重启网卡命令
sudo systemctl restart network 重启网卡
