HTML基础

什么是HTML：
  • HTML指的是超文本标记语言
  • HTML不是一种编程语言，而是一种标记语言
  • 标记语言是一套标记标签，例如'<div></div>' ‘<p></p>’
  •HTML使用标记标签来描述网页







一些简单的标签：
  •<html> ：<html>与</html>之间的文本描述网页
  •<body>：<body>与</body>之间的文本是可见的页面内容
  •<h1>这是一个标题</h1>：<h1>与</h1> 之间的文本被显示为标题
  •<p>这是一个段落</p>    ：<p>与</p>之间的文本被显示为段落







  HTML中的标签：
   •HTML标题是通过‘<h1>-<h6>’等标签进行定义的
   •HTML段落是通过‘<p>’标签进行定义的
   •HTML链接是通过‘<a>’标签进行定义的
   •HTML图像是通过‘<img>’标签进行定义的
   •HTML列表是通过‘<ul>’和'<ol>'标签进行定义的。其中‘ul’一般用来表示无序列表，‘ol’用来表示有序列表
   •HTML表格是通过‘table’ ‘thead’ ‘tbody’ ‘th’ ‘td’ 等标签来定义的

img src=“地址”   显示图片





HTML form表单：
   <form action="www.baidu.com"method="get" enctype="application/x-www-form-urlencoded"> 
               • Action属性表明表单将要提交到哪里
               • Method表示提交用到的方法分别为get个post可选
               • Enctype规定在发送表单数据前对数据进行编码的格式







输入框‘<input/>’标签最常用

   • type=“text”普通的文本输入框
   • type=“password” 带型号的密码输入框
   • type=“radio”单选框              同一类型要有统一的name属性，表明是一类+
   • type="checkbox"复选框        添加checked属性会默认选中
   • type=“file”文件上传选择
   • type=“email”带简单验证的邮箱输入
   • <texteara>标签input只能单行输入，texteara可以多行输入
    <center>         </center>    将代码输出的居中





CSS样式的引入：
     行内样式
          <p style= “display:none” > </p>

      内嵌式
           一般指<head>标签中的<style>样式

       外链式
            通过<link>标签引入外部的CSS文件

        导入外部样式
            用@import在<head>与</head>间引入

centos7 网卡配置

centos7网卡配置文件：/etc/sysconfig/network-scirpt/ifcfg-enoxxxxxx 将ONBOOT=yes是开启动态网络（DHCP）

修改静态网络：
修改BOOTPROTO=static
IPV6INIT=no
IPV6_AUTOCONF=no
ONBOOT=yes
IPADDR=192.168.7.XXX
GATEWAY=192.168.7.1
NETMASK=255.255.255.0
systemctl restart network 重启网卡

LINUX常用命令

linux命令格式：命令名+空格+参数（ls -al /temp）

pwd：当前默认的用户目录
sudo：以管理员身份运行
cd：进入一个文件夹
su：切换用户
rm -rf ：删除文件
mkdir：创建文件夹
cd .. ：返回上一目录
vim/vi+文件后缀 生成一个新文件并编辑 i：插入编辑 w：保存 q：退出 !：强制执行
clear 或者ctrl+l：清屏
cat：查看当前文件内容
ifconfig：查看当前网卡信息=ipconfig

开关机：
reboot：重启 init 6：重启 shutdown -r：重启 init 0：关机 poweroff：关机 shutdown halt poweroff：关机 logout exit：退出

文件读写操作：
touch：新建文件（ touch /tmp/hello.txt） mkdir：新建文件夹（ mkdir hello）
mkdip -p：自动逐级建立文件夹
rm：删除文件（rm hello.txt） rmdir删除文件夹（rmdir hello）
rm -rf：删除一个文件夹，文件夹内含有文件（ -r 参数逐级一个一个删除文件夹内容，-f 参数，不提示直接删除）

复制和移动：
cp：复制文件或文件夹到指定位置 mv：移动文件或文件夹到指定位置
cp修改文件时备份 改成bak文件
cp -r 拷贝文件所有文件

查看文件：
cat：显示一个文件的内容（cat /etc/passwd） more：显示一个文件的内容，并且可控（more /etc/passwd） head：显示一个文件的前几行（head/etc/passwd）（默认10行） -n 选择几行 tail：显示文件的后几行，-f 动态查看文件里新增的内容多一行显示一行（tail/etc/passwd）

搜索：
find：全盘查找文件 grep：全盘所有的文件中查找某些字符，或者在某个文件中查找 whereis：查找某个命令所在位置 which 找查找当前用户
greap -f：指定包含搜索模式的文件
bg 1：可以以一个任务在后台运行 fg 1：放到前台运行 nohup：使命令永久进行（nohup COMMAND&这样命令就可以永久在后台执行） jobs：查看所有的后台任务

：重定向 覆盖原来文件的内容 >>：不会覆盖原来文件的内容，而是追加到文件的尾部 &：后台运行符

ls -a 显示隐藏文件 ls -l 显示详细信息

压缩/解压缩文件
tar -zxvf 解压文件 tar -zcvf 压缩文件
