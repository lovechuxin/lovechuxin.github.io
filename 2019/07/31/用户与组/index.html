用户与组

useradd xxx 新建用户
passwd 修改用户密码
su 切换用户
userdel 删除用户（如果删除不了，find / -name 111 rm -rf 111 cd /var/spool/mail rm -rf 111） userdel -r 全部删除

home下可以查看用户 cat/etc/passwd/ 存储用户信息
第3位
uid唯一标识0为最高权限
•在centOS7中UID为1-999时为虚拟账户，不需要登录，傀儡账户。普通用户1000起，超级用户是0
•在centOS6中UID为1-499时为虚拟账户，不需要登录，傀儡账户。普通用户500起，超级用户是0
第4位是组gid
